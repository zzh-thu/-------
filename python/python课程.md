## 分支与函数

### 字符串string 

逗号默认后面加一个空格，int将输入的字符串转化成数字

分支和迭代|控制流 通过if语句

while与for的对比

while为无限次数的迭代而for需要已知迭代次数，都可以用break结束
while都能替代for，但for不一定能替代while

### 代码分解 使得不同的设备协同工作

分解出的模块（函数）的规则：
1.自成体系   2.使用出现异常可以跳出  3.可重用  
4.代码有组织  5.代码连贯  
函数名称、参数、函数体、返回值、文档字符串（注释）  
将形参绑定到实参，参数可以是任何形式，包括函数   
函数内部无法改变外面的变量
